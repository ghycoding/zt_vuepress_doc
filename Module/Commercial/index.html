# 首页

  1. 接口列表
描述 接口 备注
获取搜索查询条件数据 System/roleBtree 包含：项目，应用等
获取渠道 Operate_Invest/getChannel 主要用于渠道筛选查询，注意和【select-customize-new】组件的绑定关系
获取投放对比的查询条件 Operate_Invest/targetList 无
获取达人查询条件 Operate_Invest/getAuthor 无
数据报表目标完成情况接口 Operate_Invest/targetFinishList 无
投放对比时间对比接口 Operate_Invest/time_contrast 无
投放对比维度对比接口 Operate_Invest/launchComparison 无
数据报表接口 Operate_Invest/lists_new 无
数据报表自定义字段保存 Business/paramSave 无
数据报表自定义字段查询 Operate_Invest/customFieldList 数据报表table表头里面的数据都是根据后端返回进行展示的，除了冻结维度项的字段，固定的字段再同级目录下mixins/index.js中的fixedcolumn

  1. 首页模块中的【投放对比】 指标默认选择“实际消耗”；维度对比默认选择接口数据的第一项，默认选择所有项目

  2. 数据表报

    • 冻结维度里面的字段是在同级目录下mixins/index.js中的fixedcolumn中，其余字段都是接口返回
    • setdefaultcolumn() 方法设置默认的冻结维度项
    • 点击【查看助推订单】跳转到【流量助推数据】通过查询【任务ID/渠道号】获取对应的列表数据
    • 【助推分日数据】table的字段再mixins/index.js 中的 column2 存储
    • 数据报表里面的数据需要保留整数，百分比，和不需要处理的字段，这些操作都由前端进行统一处理
    • 自定义字段里面的数据也是由后端进行管理，老版本是由前端在本地写死的
